ipf解凍・作成ツールセットipf_pack_unipf(Windows版) Ver 1.00 by analysisjp
===============

某Treeofなんとかのipfファイルをなんとかするためのツール(for Windows)です。
もともとはlibipfのサンプルですが、これはバイナリ版です。
そのまま使えます。



使い方
===============

ipfpackについて
--------
ipfを作成するツールです。
指定したフォルダからのipfの作成と暗号化に対応しています。
リビジョン番号も付けられます。圧縮率を変えることもできます。

アドオンの作成や、ipfの差し替え等に使ってください。


起動するときは、zlib1.dllとlibipf.libは同じフォルダに置かないといけませんのでお忘れなく。

起動したら圧縮元フォルダを指定するテキストボックスと、IPFファイルのテキストボックスがあるので埋めましょう。
横の…をクリックすると、ファイルダイアログが出るので選択しましょう。

項目埋めたら[詳細設定]か[解凍]ボタンを押すとOKです。
[詳細設定]はIPFファイルの圧縮率、ベースリビジョン、リビジョン番号が設定できます。
圧縮率はデフォルトで大丈夫と思います。

[圧縮]は圧縮元フォルダにあるファイルを基に、IPFファイルを作成します。暗号化も行います。

他ツールで解凍できるか等のファイルの整合性などはチェックしていますが、本体クライアント
に適用した場合において、何かあっても責任は持ちませんのでその辺は注意してください。


unipfについて
--------
ipfを解凍＆復号するためのツールです

起動するときは、zlib1.dllとlibipf.libは同じフォルダに置かないといけませんのでお忘れなく。

起動したらIPFファイルと解凍先のテキストボックスがあるので埋めましょう。
横の…をクリックすると、ファイルダイアログが出るので選択しましょう。

項目埋めたら[情報のみを取得]か[解凍]ボタンを押すとOKです。
[情報のみを取得]はIPFファイルの詳細な情報が出てきます。
[解凍]は解凍先にIPFファイルを展開します。同時に復号化もします。
